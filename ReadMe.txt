== Description ==
WPX SEO Manager allows you to...

Integrate Google Analytics and Quantcast without touching your theme code.
There is also a section to add verification tokens to your header for Google Webmaster, Bing Webmaster and Alex Site Rankings.
The 3rd feature is the ability to submit your site's sitemap to Google, Bing and Ask.

[More Information](http://wpxprt.com)

== Installation ==

For an automatic installation through WordPress:

1. Go to the 'Add New' plugins screen in your WordPress admin area
2. Search for 'WPX SEO Master'
3. Click 'Install Now' and activate the plugin

For a manual installation via FTP:

1. Upload the wpx-seo-master folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area

To upload the plugin through WordPress, instead of FTP:

1. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
